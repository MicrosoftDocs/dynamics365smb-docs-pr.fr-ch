   4. <span data-ttu-id="568c1-128">Cliquez sur le bouton **OK**.</span><span class="sxs-lookup"><span data-stu-id="568c1-128">Choose the **OK** button.</span></span>  
      <span data-ttu-id="568c1-129">Les nouvelles présentations personnalisées s'affichent dans la fenêtre **Présentations état personnalisées**.</span><span class="sxs-lookup"><span data-stu-id="568c1-129">The new custom layouts appear in the **Custom Report Layouts** window.</span></span> <span data-ttu-id="568c1-130">Si une nouvelle présentation est basée sur une présentation intégrée, les termes **Copie d'une présentation intégrée** figurent dans le champ **Description**.</span><span class="sxs-lookup"><span data-stu-id="568c1-130">If a new layout is based on a built-in layout, then it has the words **Copy of a Built-in Layout** in the **Description** field.</span></span> <span data-ttu-id="568c1-131">S'il n'y avait aucune présentation intégrée pour l'état, les termes **Nouvelle présentation** figurent dans le champ **Description** de la nouvelle présentation, ce qui indique que la présentation personnalisée est vide.</span><span class="sxs-lookup"><span data-stu-id="568c1-131">If there was no built-in layout for the report, then the new layout has the words **New Layout** in the **Description** field, which indicates that custom layout is blank.</span></span>
