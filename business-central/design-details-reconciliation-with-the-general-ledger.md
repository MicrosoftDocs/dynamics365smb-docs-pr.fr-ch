<span data-ttu-id="479bc-104">Lorsque vous validez des mouvements de stock, tels que des expéditions vente, des productions ou des ajustements négatifs, les modifications de quantité et de valeur des stocks sont enregistrées respectivement dans les écritures comptables article et les écritures valeur.</span><span class="sxs-lookup"><span data-stu-id="479bc-104">When you post inventory transactions, such as sales shipments, production output, or negative adjustments, the quantity and value changes to the inventory are recorded in the item ledger entries and the value entries, respectively.</span></span> <span data-ttu-id="479bc-105">L'étape suivante de ce processus consiste à valider les valeurs de stocks dans les comptes stocks de la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="479bc-105">The next step in the process is to post the inventory values to the inventory accounts in the general ledger.</span></span>

## <a name="post-inventory-cost-to-gl-batch-job"></a><span data-ttu-id="479bc-109">Traitement par lots Valider coûts ajustés.</span><span class="sxs-lookup"><span data-stu-id="479bc-109">Post Inventory Cost to G/L Batch Job</span></span>

<span data-ttu-id="479bc-112">Les dates comptabilisation des écritures comptables sont fixées à la date comptabilisation de l'écriture valeur correspondante, sauf si l'écriture valeur se trouve dans une période comptable clôturée.</span><span class="sxs-lookup"><span data-stu-id="479bc-112">The posting dates of the general ledger entries are set to the posting date of the corresponding value entry, except when the value entry falls in a closed accounting period.</span></span> <span data-ttu-id="479bc-113">Dans ce cas, l'écriture valeur est ignorée et, vous devez modifier les paramètres comptabilité ou les paramètres utilisateur pour activer la validation dans la plage de dates.</span><span class="sxs-lookup"><span data-stu-id="479bc-113">In this case, the value entry is skipped, and you must change either the general ledger setup or the user setup to enable posting in the date range.</span></span>
