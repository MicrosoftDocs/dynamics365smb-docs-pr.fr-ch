## <a name="to-view-archived-workflow-step-instances"></a><span data-ttu-id="abcea-109">Pour afficher des instances d'étape de workflow archivées</span><span class="sxs-lookup"><span data-stu-id="abcea-109">To view archived workflow step instances</span></span>  
1.  <span data-ttu-id="abcea-110">Choisissez l'icône ![Ampoule qui ouvre la fonction Tell Me](media/ui-search/search_small.png "Dites-moi ce que vous voulez faire"), saisissez **Flux de travail**, puis sélectionnez le lien associé.</span><span class="sxs-lookup"><span data-stu-id="abcea-110">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Workflows**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="abcea-111">Ouvrez le workflow pour lequel vous souhaitez afficher toutes les instances d'étape de workflow archivées.</span><span class="sxs-lookup"><span data-stu-id="abcea-111">Open the workflow for which you want to view all archived workflow step instances.</span></span>  
3.  <span data-ttu-id="abcea-112">Dans la fenêtre **Flux de travail**, choisissez l'action **Instances d'étape de flux de travail archivées**.</span><span class="sxs-lookup"><span data-stu-id="abcea-112">In the **Workflow** window, choose the **Archived Workflow Step Instances** action.</span></span>  

    <span data-ttu-id="abcea-113">La fenêtre **Instances d'étape de workflow archivées** s'ouvre, affichant les instances d'étape du flux de travail sélectionné archivées.</span><span class="sxs-lookup"><span data-stu-id="abcea-113">The **Archived Workflow Step Instances** window opens showing the archived workflow step instances of the selected workflow.</span></span>

<span data-ttu-id="abcea-115">La fenêtre **Instances d'étape de flux de travail archivées** s'ouvre, affichant toutes les instances d'étape de flux de travail archivées.</span><span class="sxs-lookup"><span data-stu-id="abcea-115">The **Archived Workflow Step Instances** window opens showing all archived workflow step instances.</span></span> <span data-ttu-id="abcea-116">Dans le champ **Code flux de travail**, vous pouvez voir le flux de travail auquel l'instance d'étape de flux de travail a appartenu.</span><span class="sxs-lookup"><span data-stu-id="abcea-116">In the **Workflow Code** field, you can see which workflow the archived workflow step instance belonged to.</span></span>
