## <a name="to-view-archived-workflow-step-instances"></a><span data-ttu-id="47016-109">Pour afficher des instances d'étape de workflow archivées</span><span class="sxs-lookup"><span data-stu-id="47016-109">To view archived workflow step instances</span></span>  
1.  <span data-ttu-id="47016-110">Choisissez l'icône ![Ampoule qui ouvre la fonction Tell Me](media/ui-search/search_small.png "Dites-moi ce que vous voulez faire"), saisissez **Flux de travail**, puis sélectionnez le lien associé.</span><span class="sxs-lookup"><span data-stu-id="47016-110">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Workflows**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="47016-111">Ouvrez le workflow pour lequel vous souhaitez afficher toutes les instances d'étape de workflow archivées.</span><span class="sxs-lookup"><span data-stu-id="47016-111">Open the workflow for which you want to view all archived workflow step instances.</span></span>  
3.  <span data-ttu-id="47016-112">Sur la page **Flux de travail**, choisissez l'action **Instances d'étape de flux de travail archivées**.</span><span class="sxs-lookup"><span data-stu-id="47016-112">On the **Workflow** page, choose the **Archived Workflow Step Instances** action.</span></span>  

    <span data-ttu-id="47016-113">La page **Instances d'étape de flux de travail archivées** s'ouvre, affichant les instances d'étape du flux de travail sélectionné archivées.</span><span class="sxs-lookup"><span data-stu-id="47016-113">The **Archived Workflow Step Instances** page opens showing the archived workflow step instances of the selected workflow.</span></span>  
4.  <span data-ttu-id="47016-114">Sinon, choisissez l'icône ![Ampoule qui ouvre la fonction Tell Me](media/ui-search/search_small.png "Dites-moi ce que vous voulez faire"), saisissez **Instances d'étape de flux de travail archivées**, puis sélectionnez le lien associé.</span><span class="sxs-lookup"><span data-stu-id="47016-114">Alternatively, Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Archived Workflow Step Instances**, and then choose the related link.</span></span>  

<span data-ttu-id="47016-115">La page **Instances d'étape de flux de travail archivées** s'ouvre, affichant toutes les instances d'étape de flux de travail archivées.</span><span class="sxs-lookup"><span data-stu-id="47016-115">The **Archived Workflow Step Instances** page opens showing all archived workflow step instances.</span></span> <span data-ttu-id="47016-116">Dans le champ **Code flux de travail**, vous pouvez voir le flux de travail auquel l'instance d'étape de flux de travail a appartenu.</span><span class="sxs-lookup"><span data-stu-id="47016-116">In the **Workflow Code** field, you can see which workflow the archived workflow step instance belonged to.</span></span>
