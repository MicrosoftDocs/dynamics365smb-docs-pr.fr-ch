    <span data-ttu-id="84e5d-121">Les présentations d'état définissent à la fois le style et le contenu du corps de message, y compris le texte standard qui précède les informations de base relatives au document dans le corps du message.</span><span class="sxs-lookup"><span data-stu-id="84e5d-121">Report layouts define both the style and the content of the email body, including the standard text that precedes the core document information in the email body.</span></span> <span data-ttu-id="84e5d-122">Vous pouvez visualiser toutes les présentations d'état disponibles si vous choisissez le bouton **Sélectionner dans la liste complète** dans la liste déroulante.</span><span class="sxs-lookup"><span data-stu-id="84e5d-122">You can see all available report layouts if you choose the **Select from full list** button in the drop-down list.</span></span>
