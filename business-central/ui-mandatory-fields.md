# <a name="mandatory-fields"></a><span data-ttu-id="93844-103">Champs obligatoires</span><span class="sxs-lookup"><span data-stu-id="93844-103">Mandatory Fields</span></span>
<span data-ttu-id="93844-104">Lorsque vous entrez des données sur les pages dans [!INCLUDE[d365fin](includes/d365fin_md.md)], certains champs sont marqués par un astérisque rouge.</span><span class="sxs-lookup"><span data-stu-id="93844-104">When you enter data on pages in [!INCLUDE[d365fin](includes/d365fin_md.md)], certain fields are marked with a red asterisk.</span></span> <span data-ttu-id="93844-105">L'astérisque rouge signifie que le champ doit être renseigné pour terminer un processus qui utilise ce champ, par exemple, valider une transaction qui utilise la valeur du champ.</span><span class="sxs-lookup"><span data-stu-id="93844-105">The red asterisk means that the field must be filled to complete a certain process that uses the field, such as posting a transaction that uses the value in the field.</span></span>
