# <a name="welcome-to-included365acclongincludesd365acclongmdmd"></a>Bienvenue dans [!INCLUDE[d365acc_long](includes/d365acc_long_md.md)]
Si vous êtes un comptable avec plusieurs clients, vous pouvez utiliser [!INCLUDE[d365acc](includes/d365acc_md.md)] comme tableau de bord pour un meilleur aperçu de vos clients. Depuis cette fenêtre, vous pouvez accéder aux données de chaque client.  

Vous pouvez accéder à [!INCLUDE[d365acc](includes/d365acc_md.md)] en vous connectant à partir de [Dynamics 365 — Accountant Hub sur Microsoft.com](https://www.microsoft.com/en-us/dynamics365/financial-insights-for-accountants).

Dans la version actuelle de [!INCLUDE[d365acc](includes/d365acc_md.md)], vous pouvez ajouter des clients qui utilisent [!INCLUDE[d365fin_long](includes/d365fin_long_md.md)] pour gérer leurs activités.
